注意此demo用的SDK版本为2.4.2   所以你要用Xcode 7来跑这个工程。

下面是官方的原话:
Xcode 版本与 SDK 版本选择
由于 Apple 在 Xcode7 与 iOS9 中引入了新特性 Bitcode，静态库的编译需要有所调整。

如果您使用的是 Xcode6.X 版本，您可以使用 2.2.11 或 2.2.11 之前版本的 SDK；如果您使用的是 Xcode7.X 版本，您可以使用 SDK 2.3.0 或者 2.3.0 之后版本的 SDK。
